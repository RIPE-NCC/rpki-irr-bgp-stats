Adoption % is percentage of address size covered by ROAs within a region. Address size is count of unique IP addresses within aggregated prefix in a region, which can be RIR or Country.

Activation indicates number active entity within a region. An entity is active in a region if it has created certificate and hold a resource in the region, regardless whether it has created any ROAs.
